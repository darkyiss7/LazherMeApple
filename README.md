## Installation

#### CocoaPods

Vous pouvez utiliser [CocoaPods](http://cocoapods.org/) pour installer les librairies en les ajoutant au fichier `Podfile`:

```ruby
platform :ios, '15.3

use_frameworks!

pod 'Firebase/Core'
pod 'Firebase/Auth'
pod 'Firebase/Database'

```
